My Journey

How I learned to make services talk

// 01

First API Call

My first real API call was to a weather service — a simple GET request that returned JSON. Seeing live data come back from a server I didn't control felt like unlocking a superpower. Suddenly the entire internet became something I could interact with programmatically.

// 02

Going Deeper

I started learning the patterns — authentication headers, rate limiting, pagination, error codes, retry logic. I built wrappers around APIs to make them clean to use in my projects. Understanding how to handle failures gracefully is what separates a working integration from a production one.

// 03

Building for Clients

Today I build full integration layers for clients — connecting CRMs to email platforms, syncing data between services, building webhooks that react to real-time events. APIs are how modern businesses run, and I know how to wire them together reliably.
