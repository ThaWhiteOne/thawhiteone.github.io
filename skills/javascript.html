My Journey

How JS became my frontend weapon

// 01

Starting with the DOM

My first JS experience was manipulating the DOM — making things appear, disappear, animate. It felt like magic. Unlike Python which runs on a server, JavaScript runs right in front of the user, making instant feedback possible. That immediacy was addicting.

// 02

Discovering React

When I found React, everything clicked. Instead of thinking about DOM mutations, I started thinking in components and state. Build a Button once, use it everywhere. Change state, UI updates automatically. It made complex UIs manageable and gave my code real structure.

// 03

Full-stack thinking

Today I use JavaScript and React to connect everything together. API calls, real-time updates, complex state management, animations — the frontend is where users live, and I treat it with the same care as any backend system. Performance and feel matter as much as functionality.
